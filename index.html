Fall detection application

The project which I have been assignment from tallaght university hospital is a system which can be used by dementia patients which are currently living at home. Which will detect if the patent has fallen over in their home. I must have an application which can detect 3D movement. I believe the best solution which I will start develop would be an application for smart watches, which will be linked to an application on a handheld device. Using the accelerometer on the smartwatch to detect if a fall has taken place and if it has been sending a message to the handheld device. Upon receiving a message the phone will make a call to the eldery persons family if there is no response a call will be sent straight to the emergency services for an ambulance. The message must be real for the emergency services to react so in the handheld application using the microphone the user must first create a voice message which will be played when the call is made to the emergency services. This part must be set up when the application is downloaded and would have to be done by the patient's family.
